INTRODUCTION
------------
 
Shell script to dump PostgreSql database


INSTALLATION/ TEST/ EXECUTION
-----------------------------
	
 * ./dump_database.sh [env] [schema]
	-> env defines the environment [dev|uat|prod]
	-> schema defines whether only schema or data
